# School Attendance Manager (C Project)

##  Description

This is a terminal-based Student Attendance Management System written in C using file handling. It allows efficient management of student records and attendance tracking.

##  Features

* Add Student
* Mark Attendance
* Show Attendance Report
* Students Below 75% Attendance
* Class Average Attendance
* Delete Student

##  Technologies Used

* C Programming
* File Handling

##  How to Run

### Compile:

gcc program.c -o attendance

### Run:

./attendance

##  Data Storage

* `attendance.txt` is used to store student records
